The main folder is the `app` folder.
    Inside the `app` folder you can see the `model`, `view` and `controller` folders.
    There is also a `javascript` folder all the `react` UI components will be inside this folder.

    In `views` you can see the `home` folder with a `home_controller.rb` on `controller` folder.
        In home folder there is a file `index.html.erb` this will be the main file for the UI.
            <%= javascript_pack_tag "index" %>
            means that it will include `Tile.js` from `javascript/packs/Tile.js`
